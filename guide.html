Usage

Layout

For grid layout, place section-elements inside a main-element.

To force a section to span the full row, use class "full" on the section element.

        <main>
  <section>
          content here
  </section>
</main>

Page main header

The main header of the page must be inside of a head-element.

        <header>
<h1>Page header here!</h1>

Navigation

To get a responsive navigation bar with sub navigation menus, use the following html structure:

        <header>
  <nav>
    <ul>
          <li><a href="href">Nav item 1</a></li>
          <li><a href="href">Nav item 2</a></li>
          <li><a href="href">Nav item 3</a></li>
          <li><a href="href">Sub navigatneion</a>
            <ul>
              <li><a href="href">Sub nav item 1</a></li>
              <li><a href="href">Sub nav item 2</a></li>
            </ul>
          </li>
    </ul>
  </nav>
</header>

Loading indicator

For a loading indicator, place a div with class "loading". To display text in the loading indicator, place a p-element inside the div.

        <div class="loading">
  <p>loading...</p>
</div>

For a fullscreen loading indicator, add class "full" to the div.

Messages and Warnings

There are currently one type of message and one type of warning

Use a div for theese with the right type of class. The text in the boxes should be within the div. 

		<div class="warning">This is a warning. Text in the warning should be placed here</div> 

<div class="message">This is a message. The text in the message goes here</div>

List with objects

This list of objects is used with divs. This is a flexbox.

One div with the classname"objects" to specify the whole list. All underlying divs will take one space in the list.

			<div class="objects">
	<div class="you can put styled objects here">This is object one. if you want text in the object. write it here.</div>
	<div>This is text for objet two.</div>
	<div>This is text for object three.</div>
</div>

Text

There is many different styles of text. They all use paragraf as their element with a specified classname.

For normal text just like this, dont use any classname.

			<p>normal text goes in here</p>
<p class="textImportant">text with border bottom and shadows goes in here</p>
<p class="textAnimation">moving text goes in here</p>
<p class="smallText">small text here</p>
<p class="storaBokstaver">text here translates to only capital letters</p>
<p class="kapow">animated text here</p>
<p class="speech">a speechbubble around this text</p>
<p class="thought">thought bubbles around this text</p>

Panels

Panels are div element with a specific class. All panels use the "panel" class. This gives a neutral look.

for negative and positive panels you also have to specify their specific class.

			<div  class="panel">Text for a neutral panel here</div>
<div  class="panelPositive panel">Text for a positive panel here</div>
<div  class="panelNegative panel">Text for a negative panel here</div>

Tables

Two different types of tables. Both use table element. Use correct classname for the table to get the right styling.

table class "longTable" is used for only one column while table class "multipleColumns" is styled for many columns

One column table:

			<table class="longTable">
	<thead>
		<tr>
			<th>Table header goes here</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>Data goes here</td>
		</tr>
	</tbody>
  <tfoot>
		<tr>
			<td>Extra information about table goes here</td>
		</tr>
	</tfoot>
</table>

Multiple columns table:

		<table class="multipleColumns">
	<thead>
		<tr>
			<th>Column 1 header goes here</th>
			<th>Column 2 header goes here</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>Data for row 1, column 1 goes here</td>
			<td>Data for row 1 column 2 goes here</td>
		</tr>
		<tr>
			<td>Data for row 2 column 1 goes here</td>
			<td>Data for row 2 column 2 goes here</td>
		</tr>
	</tbody>
	<tfoot>
		<tr>
			<td>Extra information about table goes here</td>
		</tr>
	</tfoot>
</table>

Forms

Form element for sending data. Styles legend of fieldset. Styles for text input, select, radio buttons and buttons.

This exampel below collects all above types within one form but multiple fieldsets. You will change fieldsets and form to your needs.

			<form>
	<fieldset>
		<legend>Header for the fieldset</legend>
		<input type="text" placeholder="Text to show in textarea while the area is empty"/>
		<input type="text" placeholder="Text to show in a disabled(grey) textarea" disabled/>
		<textarea placeholder="Text to show in a textarea before while area is empty.
		this textarea is scrollable up and down. and text starts att a new row if needed"></textarea>
    </fieldset>

	<fieldset>
		<legend>Header for the fieldset</legend>
		<select>
			<option>Text for option one</option>
			<option>Text for option two</option>
		</select>
    </fieldset>

    <fieldset>
		<legend>Header for the fieldset</legend>
		<input type="radio" name="radio1" value="value 1" checked> <label class="radioLabel">Text for the radiobuttons label one here</label>
		<input type="radio" name="radio1" value="value 2"> <label class="radioLabel">Text for the radiobuttons label two here</label>
    </fieldset>
    <fieldset>
		<legend>Header for the fieldset</legend>
		<button class="standard" name="standardButton">Text on the standardbutton</button>
		<button class="function" name="functionButton">Text on the functionButton</button>
		<button class="positive" name="positiveButton">text on the positiveButton</button>
		<button class="negative" name="negativeButton">text on the negativeButton</button>
		<input type="submit" value="Button to submit the form. Text on the button here">
    </fieldset>
</form>
